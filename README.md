🚀 Internet Speed Test Application

This project is a web-based Internet Speed Test tool developed using Python, Streamlit, and Speedtest CLI. It provides users with real-time internet performance metrics including download speed, upload speed, and network latency (ping) through a clean and interactive user interface.

The application automatically identifies the optimal test server, performs network diagnostics, and presents the results in an easy-to-understand dashboard. Users can also download test reports in JSON format for record keeping and analysis.

✨ Features
Real-time Internet Speed Testing
Download Speed Measurement (Mbps)
Upload Speed Measurement (Mbps)
Ping/Latency Monitoring
Automatic Best Server Detection
Interactive Streamlit Dashboard
JSON Report Generation and Download
User-Friendly Interface
Cross-Platform Support
